# stock-pvp-price-calculator
API to calculate the P/BV ratio (Price-to-Book Value) of Brazilian stocks and REITs, with smart MongoDB caching and real-time market data powered by Brapi. 

PT: API para calcular o P/VP (Preço sobre Valor Patrimonial) de ações e FIIs brasileiros, com cache inteligente via MongoDB e dados em tempo real via Brapi.
